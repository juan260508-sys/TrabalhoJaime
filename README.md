# 🚀 Orbix — E-commerce de Tecnologia

O **Orbix** é um projeto de e-commerce focado na venda de produtos de tecnologia, desenvolvido como trabalho facultativo por alunos do **1º semestre de Sistemas de Informação**.

A proposta do projeto é aplicar, na prática, conceitos iniciais de desenvolvimento web, criando uma aplicação funcional, organizada e com uma boa experiência para o usuário.

---

## 📌 Sobre o Projeto

O Orbix simula uma loja virtual onde é possível visualizar e pesquisar produtos tecnológicos, como:

* 🎧 Fones de ouvido
* 🎤 Microfones
* ⌨️ Teclados mecânicos
* 🖥️ Monitores
* 🔊 Caixas de som
* 🔋 Power banks
* 🖨️ Impressoras

O sistema foi desenvolvido com foco em simplicidade e aprendizado, utilizando manipulação de DOM e lógica de programação em JavaScript.

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---

## ⚙️ Funcionalidades

* 🔍 Sistema de busca de produtos
* 📦 Listagem dinâmica de produtos
* 💰 Exibição de preços
* 🖼️ Renderização de imagens dos produtos
* ❌ Mensagem de erro ao não encontrar resultados

---

## 🎯 Objetivos do Projeto

* Praticar lógica de programação
* Aprender manipulação do DOM
* Entender a estrutura de um e-commerce
* Desenvolver organização de código
* Trabalhar com dados em formato JSON
